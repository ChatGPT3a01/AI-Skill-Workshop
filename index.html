Day 1 課程簡報

Course Slides

本課程將帶你從零開始，學會把「你的專業流程」轉化為「AI 可重複執行的技能模組」。從概念理解、結構拆解、實作撰寫，到跨平台部署與流程變現，完整走過一次 AI Skill 的打造歷程。
Part 1
AI Skill 基礎概念
什麼是 Agent Skills？工作流程拆解方法
Part 2
SKILL.md 結構與實作
YAML 格式、四大關鍵指令、動手寫第一個 Skill
Part 3
在 Antigravity 執行 Skill
免費平台實測、詢價與客訴測試、除錯技巧
Part 4
Claude Code 與 CLI 跨平台
API Key 設定、Gemini CLI、三平台對照、同步腳本
Part 5
安装 Claude Code Skills
詳細步驟、16 個官方 Skills、驗證安装、常見問題
Part 6
進階技巧與流程變現
防呆機制、AI 生成器、流程變現五層架構
Part 7
實戰專案與成果展示
會議紀錄產生器、Day 1 成果展示
Part 8
Pencil AI 設計工具
用 Skill 驅動 AI 做設計、MCP 協定、Design to Code

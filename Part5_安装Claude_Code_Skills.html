📦 什麼是 Claude Code Skills？

定義

Claude Code Skills 是由 Anthropic 官方提供的特化功能包，用來擴展 Claude Code 的能力。

Skills 包含的內容

  • 預制的提示詞：針對特定任務優化的指令
  • 工作流範本：可重複使用的流程和最佳實踐
  • 配置文件：針對特定工具的設置
  • 工具整合：與 Claude Code 配合使用的工具鏈

為什麼需要安装 Skills？

  • 🚀 提升效率：無需從零開始編寫配置
  • 📚 學習最佳實踐：官方推薦的工作流程
  • 🔗 快速集成：與其他工具無縫配合
  • ✨ 持續更新：獲取最新功能和改進

✅ 安装前檢查清單

在開始安装之前，請確保您已經完成以下準備：

1
已安装 Claude Code CLI

您應該已經通過 Part 4 安装了 Claude Code。如果還未安装，請先完成 Part 4 的學習。

檢查方式： 在終端中輸入 claude --version

2
安装了 Git

Skills 需要通過 Git 克隆官方倉庫。如果您還沒安装 Git，請先安装。

檢查方式： 在終端中輸入 git --version

3
了解終端基本操作

您需要能夠在終端中執行基本命令（如 cd、ls/dir 等）。

⚠️ 注意： 如果上述任何一項您還未完成，請先完成相應的準備工作，再繼續本章節。

🖥️ 第一步：打開終端

根據您使用的作業系統，選擇相應的方法：

Windows 用戶

方法 1：使用 Windows 終端（推薦）

  1. 按下 Win + X
  2. 從列表中選擇 終端 (T) 或 Windows 終端

方法 2：使用 PowerShell

  1. 按下 Win + R
  2. 輸入 powershell 並按 Enter

方法 3：使用命令提示字元

  1. 按下 Win + R
  2. 輸入 cmd 並按 Enter

macOS 用戶

  1. 按下 Cmd + Space 打開 Spotlight
  2. 輸入 terminal
  3. 按 Enter 打開 Terminal

Linux 用戶

根據您的發行版本，通常可以通過：

  • 按下 Ctrl + Alt + T（Ubuntu/Debian）
  • 或在應用程式選單中搜尋「Terminal」或「終端」
✅ 成功標誌： 您應該看到一個黑色或白色的窗口，其中有一個提示符（如 $ 或 > ），等待您輸入命令。

📂 第二步：導航到工作目錄

現在我們需要導航到一個合適的位置來安装 Skills。

為什麼需要選擇目錄？

Skills 需要被克隆到您電腦上的某個位置。最好的做法是創建一個專門的目錄來管理所有 Skills。

建議的目錄位置

Windows 用戶

在您的用戶目錄下創建一個 claude-skills 文件夾：

C:\Users\您的用戶名\claude-skills

macOS 和 Linux 用戶

在您的主目錄下創建一個 claude-skills 文件夾：

~/claude-skills

詳細步驟

1
查看當前位置

在終端中輸入以下命令來查看您現在所在的目錄：

pwd

預期輸出： 會顯示當前目錄的完整路徑

2
創建工作目錄

輸入以下命令創建一個新的目錄：

mkdir claude-skills

說明： 這會在當前位置創建一個名為 claude-skills 的文件夾

3
進入新目錄

輸入以下命令進入您剛建立的目錄：

cd claude-skills

驗證： 您應該在終端中看到目錄名稱已更改

💡 提示： 您可以通過在終端中輸入 ls（Mac/Linux）或 dir（Windows）來列出當前目錄中的文件。

📥 第三步：克隆 Anthropic Skills 倉庫

現在我們將從 GitHub 下載官方的 Skills。

什麼是「克隆」？

克隆（Clone）是 Git 的一個功能，它會將遠端倉庫（GitHub 上的代碼）複製到您的本地電腦上。

克隆命令

在終端中複製並執行以下命令：

git clone https://github.com/anthropics/skills.git

步驟分解

  1. git clone - Git 命令，用於克隆倉庫
  2. https://github.com/anthropics/skills.git - GitHub 倉庫的完整 URL

執行結果

在您按 Enter 後，終端會：

  • 📥 從 GitHub 下載所有 Skills 文件
  • ⏳ 這可能需要 1-2 分鐘，具體取決於您的網路速度
  • ✅ 完成後會返回到命令提示符
  • 📂 在當前目錄中創建一個名為 skills 的文件夾
💡 耐心等待： 第一次克隆可能較慢。請不要中斷操作，讓它完成下載。

📂 第四步：進入 Skills 目錄

現在我們進入已克隆的 Skills 文件夾。

1
進入 skills 目錄

輸入以下命令：

cd skills
2
進入 skills 子目錄

GitHub 倉庫中的實際 Skills 文件在 skills 子文件夾內：

cd skills

現在您應該在： claude-skills/skills/skills

驗證位置

在終端中輸入以下命令查看當前目錄中的內容：

ls

在 macOS/Linux 上 或

dir

在 Windows 上

預期看到的 Skills 列表

您應該能看到以下文件夾（至少其中一些）：

  • algorithmic-art
  • brand-guidelines
  • canvas-design
  • doc-coauthoring
  • docx
  • ...等等
✅ 成功標誌： 如果您能看到這些 Skills 文件夾，表示克隆成功！

⚙️ 第五步：安装 Skills 到 Claude Code

這是關鍵步驟：將下載的 Skills 安装到 Claude Code。

安装方法

在確認您在 skills 目錄中後，執行以下命令：

for dir in */; do claude install "$dir"; done
⚠️ Windows 用戶注意： 上面的命令是 bash 語法。如果您使用的是 Windows 命令提示字元（cmd），請使用以下命令： 
for /d %G in (*) do claude install "%G"

或者如果您使用 PowerShell：

Get-ChildItem -Directory | ForEach-Object { claude install $_.Name }

安装過程說明

這個命令會：

  1. 掃描當前目錄中的所有文件夾
  2. 對每個文件夾執行 claude install 命令
  3. 逐個安装所有 Skills 到 Claude Code

安装期間會發生什麼？

  • ⏳ 終端會顯示每個 Skill 的安装進度
  • ✅ 每個成功安装的 Skill 會顯示完成消息
  • 📝 某些 Skills 可能會要求您確認或輸入配置
  • ⏱️ 整個過程通常需要 2-5 分鐘
💡 提示： 安装過程中，終端可能會變得非常繁忙。這是正常的！請耐心等待所有 Skills 安装完成。

🚀 如何使用已安装的 Skills？

在 Claude Code 中使用 Skills

現在您已經安装了所有 Skills，可以在 Claude Code 中使用它們。以下是基本用法：

方法 1：直接指定 Skill

在使用 Claude Code 時，您可以明確指定要使用的 Skill：

claude --skill pdf 处理我的 PDF 文件

方法 2：讓 Claude 自動選擇

Claude 也可以根據您的任務自動選擇適當的 Skill：

claude 幫我創建一個 PowerPoint 演示文稿

Claude 會自動使用 pptx Skill。

方法 3：組合多個 Skills

對於複雜任務，Claude 可以同時使用多個 Skills：

claude 從 PDF 中提取數據並在 Excel 中創建報表

這會結合使用 pdf 和 xlsx Skills。

查看 Skill 詳情

要了解特定 Skill 的詳細用法：

claude skill-info 技能名稱

例如：

claude skill-info docx

❓ 常見問題與解決方案

❌ 問題 1：「命令未找到」錯誤

錯誤消息示例：

bash: claude: command not found

解決方案：

  • ✓ Claude Code 未正確安装。請重新檢查 Part 4 的安装步驟
  • ✓ 嘗試完全關閉並重新打開終端
  • ✓ 重新啟動您的電腦

❌ 問題 2：克隆倉庫失敗

錯誤消息示例：

fatal: unable to access 'https://github.com/...'

解決方案：

  • ✓ 檢查您的網絡連接
  • ✓ 確保您的防火牆未阻止 GitHub 訪問
  • ✓ 等待幾分鐘後重試

❌ 問題 3：Skills 未顯示在列表中

解決方案：

  • ✓ 確認您在正確的目錄中執行安装命令
  • ✓ 嘗試手動安装單個 Skill：claude install ./pdf
  • ✓ 檢查 Claude 的配置文件是否可寫

❌ 問題 4：权限拒绝错误

錯誤消息示例：

Permission denied

解決方案：

  • ✓ 使用 sudo（macOS/Linux）：sudo claude install ./skill-name
  • ✓ 以管理員身份運行終端（Windows）

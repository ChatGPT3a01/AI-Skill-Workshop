## 📖 課程簡介

> 不只是學 Skills，而是學會把「你的專業流程」轉化為「AI 可重複執行的技能模組」。

Day 1 帶你從零開始，完整走過 AI Skill 的打造歷程：**概念理解 → 結構拆解 → 實作撰寫 → 跨平台部署 → 流程變現**。課程結束時，你將帶走至少 **2 個可執行的 AI Skill** + 完整的 Skill 專案資料夾。

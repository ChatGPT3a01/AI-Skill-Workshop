## 📖 課程簡介

> 不只是學 Skills，而是學會把「你的專業流程」轉化為「AI 可重複執行的技能模組」。

兩天課程帶你從零開始，完整走過 AI Skill 的打造歷程：

- **Day 1**：打造第一個 AI Skill — 概念理解 → 結構拆解 → 實作撰寫 → 跨平台部署 → 流程變現
- **Day 2**：從單一 Skill 擴展為多 Skill 工作系統 — Skill Chain → 自動化 → Firebase Skills → 企劃行銷實戰

│
├─► 轉為 SKILL.md ────► 📄 第三層：AI 可執行的技能

├─► 加上表單 ─────────► 🖱️ 第四層：任何人都能用的生成器
│

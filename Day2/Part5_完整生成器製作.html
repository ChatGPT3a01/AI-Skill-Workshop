G-1｜生成器設計理念

比較項目Day 1 的做法Day 2 的進化
使用者自己（懂 SKILL.md）整個團隊（不需懂 AI）
操作方式開 AI 平台、貼指令開瀏覽器、填表單、點按鈕
Skill 入口Antigravity / Claude Code CLI網頁生成器（skill-generator.html）
維護難度需手動更新每個 Skill集中在一個 HTML 檔案管理
分享方式複製 SKILL.md 給對方傳一個連結（或 HTML 檔案）
核心理念：讓同事只需「開網頁 → 選功能 → 填表單 → 產生 → 複製」，完全不需要知道 SKILL.md 是什麼！

G-1｜生成器使用流程（5 步）

Step 01

開啟網頁

瀏覽器打開 skill-generator.html，不需要安裝任何軟體

Step 02

選擇功能

點選頁面頂部的功能標籤：Email 回覆 / 會議紀錄 / 提案報告

Step 03

填寫表單

在對應欄位填入情境資訊、選擇語氣與優先度

Step 04

點擊產生

按下「產生 Prompt」按鈕，系統自動組合最佳指令

Step 05

複製使用

複製產生的 Prompt，貼入任何 AI 平台立即使用

G-2｜JavaScript 切換與產生邏輯

// Tab 切換函式
function switchTab(tabName) {
  document.querySelectorAll('.skill-panel').forEach(p => {
    p.classList.remove('active');
  });
  document.querySelectorAll('.tab').forEach(t => {
    t.classList.remove('active');
  });
  document.getElementById('panel-' + tabName).classList.add('active');
  event.target.classList.add('active');
  currentTab = tabName;
}

// Email Prompt 產生函式
function generateEmailPrompt() {
  const content  = document.getElementById('email-content').value;
  const context  = document.getElementById('email-context').value;
  const priority = document.querySelector('input[name="priority"]:checked').value;
  const tone     = document.querySelector('input[name="tone"]:checked').value;

  const priorityMap = {
    maintain_schedule: '維持原定時程，說明原因',
    negotiate:         '尋求雙方都能接受的折衷方案',
    accept:            '接受對方要求，但提出配套條件'
  };
  const toneMap = {
    formal:    '正式、專業的商務語氣',
    friendly:  '親切友善、保持良好關係',
    assertive: '積極主動、展現執行力'
  };

  return `你是一位專業的商務溝通顧問。

【收到的 Email】
${content}

【我方背景與立場】
${context}

【處理目標】：${priorityMap[priority]}
【回覆語氣】：${toneMap[tone]}

請撰寫一封完整的回覆 Email，包含主旨與正文。`;
}

## 📖 課程簡介

> Day 1 學會打造一道菜，Day 2 學會經營一間餐廳。

Day 2 從「多 Skill 思維」出發，帶你設計 Skill Chain 串接、跨平台部署、NotebookLM 自動化、Firebase Agent Skills 實戰，最終完成「一句話生成行銷企劃書 + 自動轉 Word」的完整工作系統。

## 🧠 核心觀念

```
Day 1：學會做一道菜（單一 Skill）
│
Day 2：學會經營一間餐廳（多 Skill 系統）
│
├─► Skill Chain 串接 ────► 多步驟自動化
│
├─► 跨平台部署 ─────────► 一份 SKILL.md 三平台通用
│
├─► NotebookLM 自動化 ──► Python + Playwright + Flask
│
├─► Firebase Agent Skills ► Google 官方 Skill 套件
│
└─► 範本驅動設計 ────────► 範本 + 佔位符 + SKILL.md = 文件自動化
```

H-2｜README 快速開始範本

# 📦 My Skill Project

我的專業 AI Skill 專案包，包含 Email 回覆、會議紀錄、提案報告三大功能。

## ⚡ 快速開始

### 方式一：Antigravity（推薦）
將 `.agent/skills/` 資料夾放到你的專案目錄，關鍵字即可觸發

### 方式二：Claude Code CLI
將 `.claude/skills/` 資料夾放到專案目錄，直接對話觸發

### 方式三：網頁表單
開啟 `skill-generator.html`，填表、產生、複製，不需安裝任何軟體

## 📋 Skills 清單

| Skill 名稱 | 觸發關鍵字 | 功能說明 |
|------------|----------|----------|
| email-reply-helper | Email、回覆 | 自動分析 Email 並產生專業回覆 |
| meeting-notes-generator | 會議、紀錄 | 將會議逐字稿轉為結構化會議紀錄 |
| proposal-generator | 提案、報告 | 根據目標與資料自動生成提案報告 |

H-4｜Git 版本控制

# 初始化 Git 儲存庫
git init
git branch -M main

# 建立 .gitignore
echo ".DS_Store" > .gitignore
echo "*.log" >> .gitignore
echo "node_modules/" >> .gitignore

# 首次提交
git add README.md CLAUDE.md skill-generator.html
git add .agent/ .claude/ scripts/ examples/

H-5｜分享給團隊的三種方式

🔗 GitHub 分享

一個連結，團隊成員 git clone 後立即可使用所有 Skill，最適合小團隊

📦 ZIP 壓縮包

將資料夾打包為 .zip，適合非技術同事，不需安裝 Git 即可使用

🌐 Netlify 網頁

一鍵部署到網路，團隊只需一個連結就能使用網頁生成器

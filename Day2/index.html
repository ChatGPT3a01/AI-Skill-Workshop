Day 2 課程簡報

Course Slides

Day 1 你學會做「一道菜」，Day 2 你要學會開「一間餐廳」——把多個 Skill 串成完整的 AI 工作系統。今天你將帶走：一套 Skill 工作系統 + AI 助理設定檔 + 可上線分享的網頁 + 個人實戰作品。
Part 1
Day 1 回顧 + 多 Skill 思維
快速複習 + 一道菜 vs 一間餐廳 + 三大拆解原則
Part 2
Skill 群組 + 工作鏈串接
建立客戶提案 3-Skill 群組 + 三種串接模式 + 流程編排 Skill
Part 3
AI 助理設定檔 CLAUDE.md
角色定義 + 三大載體 + 實作專屬 AI 助理人設
Part 4
跨平台實測 + Skill 生態系
三平台對照實驗 + 上午 KPT 反思 + 生態系五層架構
Part 5
完整生成器製作
多功能三合一 Skill 工作台 + 一鍵產生提示詞
Part 6
打包分享 + Netlify 上架
標準資料夾結構 + README + 三種方法部署到 Netlify
Part 7
完整流程實戰示範
PPTX / XLSX / DOCX / NotebookLM / PDF — 真實輸出六大工作流
Part 8
Skill 優化與總結
PDCA 持續優化 + 版本管理 + 十大核心觀念總整理
Part 9
Script 自動化進階
用 Python Script 讓 Skill 自動化批次執行
Part 10
NotebookLM 自動化
MCP Server + Skill 打造 NotebookLM 全自動化工作流
Part 11
Firebase Agent Skills
Google 官方 Agent Skills 安裝與實戰
Part 12
企劃行銷 Skill 實戰
範本 + SKILL.md 自動生成行銷企劃書
